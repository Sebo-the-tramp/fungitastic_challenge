Fine-grained semantic segmentation is inherently challenging, as it requires both precise localization and the ability to distinguish subtle inter-class differences. This challenge is further amplified by the highly imbalanced class distribution of the FungiTastic dataset. A straightforward solution is to prompt SAM3 with class-specific queries; however, this approach suffers from limited scalability, slow inference, and low IoU due to frequent class omissions. Additionally, its computational cost increases with both the number of images and the number of classes.

To address these limitations, we propose a two-stage framework that decouples segmentation from classification. In the first stage, SAM3 is employed to generate class-agnostic through macro taxonomy queries. In the second stage, a foundation model (DINOv3) extracts latent representations for each segment, enabling classification in embedding space. This design improves both scalability and flexibility, while reducing dependence on class-specific prompts.

We report baseline results across one-shot, few-shot, and fully supervised settings, maintaining robustness under varying levels of supervision. Furthermore, we show that leveraging the extracted segments for data augmentation in the few-shot regime can further improve performance.

The code will be publicly released upon acceptance.
