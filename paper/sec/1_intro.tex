\section{Introduction}

Fine-grained semantic segmentation requires both precise localization and the ability to distinguish subtle inter-class differences. This challenge is particularly pronounced in domains such as mycology, where classes exhibit minimal visual differences and long-tailed distributions.

Foundation models such as SAM3 \cite{carion2025sam3segmentconcepts} have demonstrated strong class-agnostic segmentation capabilities, and can be adapted to semantic segmentation via class-specific prompting. However, as also noted in prior work \cite{carion2025sam3segmentconcepts}, this approach struggles with fine-grained concepts. Since the FungiTastic dataset \cite{Picek_2025_CVPR} provides a small subset of segmented mushrooms, we confirm this limitation by prompting SAM3 with specific scientific names (e.g., Cantharellus cibarius, Boletus edulis) showing that this approach leads to incomplete segmentations and low performance. Even under oracle classification, we observe a mean IoU of only 59\% (Table~\ref{tab:results}). Moreover, this approach scales poorly, as it requires multiple forward passes per image (one for each class, when correct label is not known a priori) resulting in significant computational overhead.

Motivated by these limitations, we propose a two-stage framework that decouples segmentation from classification. We first extract class-agnostic segments using SAM3 \cite{carion2025sam3segmentconcepts}, and then assign semantic labels by embedding each segment through a vision foundation model (DINOv3 \cite{simeoni2025dinov3}). This design eliminates the need for class-conditioned inference, improving both efficiency and scalability, while naturally supporting low-data regimes.

Finally, we investigate whether segment-level representations can be leveraged beyond classification, specifically as a source of data augmentation in few-shot settings.

\textbf{Contributions}.
(i) We empirically validate the limitations of prompt-based SAM3 for fine-grained segmentation, showing poor IoU even under oracle classification and unfavorable scaling with the number of classes.
(ii) We propose a simple and scalable two-stage framework that decouples segmentation and classification via segment embeddings extracted using macro taxonomy queries.
(iii) We benchmark multiple backbone models and evaluate segment-based augmentation strategies, providing insights into their effectiveness in low-data regimes.
